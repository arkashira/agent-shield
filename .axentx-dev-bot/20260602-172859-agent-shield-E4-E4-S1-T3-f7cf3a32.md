## Final Synthesized Deployment Guide for agent-shield

### 1. Prerequisites
| Tool | Minimum Version | Notes |
|------|-----------------|-------|
| Python | 3.11 | For local dev (optional) |
| Docker | 24.x | For building & running containers |
| Docker‑Compose | 2.x | For local multi‑service orchestration |
| kubectl | 1.28+ | For Kubernetes deployment |
| Helm | 3.12+ | Optional, for Helm chart usage |

### 2. Building the Docker image

```bash
# From the project root
docker build -t axentx/agent-shield:latest .
```

### 3. Running locally with Docker‑Compose

1. Copy the example config and edit it:

```bash
cp config.example.yaml config.yaml
# edit config.yaml with your values
```

2. Start the service:

```bash
docker compose up -d
```

The API will be available at `http://localhost:8000`.

### 4. Deploying to Kubernetes

A minimal Helm chart is provided in `charts/agent-shield`.  
Install it with:

```bash
helm repo add axentx https://axentx.github.io/charts
helm install agent-shield axentx/agent-shield --values values.yaml
```

`values.yaml` should contain the same keys as `config.example.yaml`.

### 5. Configuration

`config.yaml` is the single source of truth for runtime settings.  
Key sections:

| Section | Key | Description |
|---------|-----|-------------|
| `security` | `api_key` | Secret key for API authentication |
| `memory` | `backend` | Memory backend (`redis`, `sqlite`, etc.) |
| `tools` | `enabled` | List of tool names to expose |
| `logging` | `level` | Log level (`debug`, `info`, `warn`, `error`) |

All values are optional; defaults are loaded from the codebase.

### 6. Testing the deployment

```bash
# Health check
curl http://localhost:8000/health
# Expected response: {"status":"ok"}
```

### 7. CI/CD

Add the following to your GitHub Actions pipeline:

```yaml
- name: Build image
  run: docker build -t axentx/agent-shield:${{ github.sha }} .

- name: Push image
  run: |
    echo ${{ secrets.DOCKERHUB_TOKEN }} | docker login -u ${{ secrets.DOCKERHUB_USER }} --password-stdin
    docker push axentx/agent-shield:${{ github.sha }}
```

---

Happy deploying! 🚀
