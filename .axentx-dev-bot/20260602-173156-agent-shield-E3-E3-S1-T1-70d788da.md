```python
import psutil
import logging
import time
from config import MEMORY_LIMIT, MEMORY_THRESHOLD, ALERT_CHANNEL

logger = logging.getLogger(__name__)

def get_memory_usage():
    process = psutil.Process()
    return process.memory_info().rss / (1024 * 1024)  # in MB

def get_top_memory_functions():
    import line_profiler
    profiler = line_profiler.LineProfiler()
    profiler.add_function(get_memory_usage)
    profiler.run("get_memory_usage()")
    profiler.print_stats()
    return profiler.get_line_stats()[:5]

def run_profiler():
    memory_usage = get_memory_usage()
    if memory_usage > MEMORY_THRESHOLD * psutil.virtual_memory().total / (1024 * 1024):
        logger.warning(f"Memory usage exceeded {MEMORY_THRESHOLD * 100}% threshold")
        # Send alert to Ops channel
        send_alert(ALERT_CHANNEL, f"Memory usage exceeded {MEMORY_THRESHOLD * 100}% threshold")
    top_functions = get_top_memory_functions()
    logger.info(f"Top 5 memory-heavy functions: {top_functions}")
    # Log top 5 memory-heavy functions
    log_top_functions(top_functions)

def schedule_profiler():
    while True:
        run_profiler()
        time.sleep(600)  # 10 minutes
```

```python
def main():
    schedule_profiler()

if __name__ == "__main__":
    main()
```

## 1. Acceptance Criteria
1. **Periodic profiling** – The profiler executes automatically every 10 minutes (±5 seconds) without manual trigger.  
2. **Peak memory capture** – For each run it records the *peak* RSS memory of the agent process and persists the value to `memory_profiler.log`.  
3. **Threshold alert** – If the recorded peak exceeds **80 %** of the `AGENT_MEMORY_LIMIT_MB` environment variable, an alert message is posted to the configured Ops Slack channel (or fallback webhook).  
4. **Top‑5 functions** – The profiler logs the five functions (module + line) that contributed the most to the resident set size, ordered by descending memory consumption.  
5. **Zero‑impact on normal operation** – The profiling thread must not block the main agent loop; CPU overhead stays below 2 % on a idle agent (verified by a benchmark test).  
6. **Graceful degradation** – If `psutil` cannot be imported or the OS does not expose memory maps, the profiler disables itself and writes a warning to the log, without raising an exception.
