### 1. Acceptance Criteria
1. **Retry Count** – A tool call that returns a transient error (HTTP 5xx or timeout) is automatically retried **up to `MAX_RETRIES`** times (default = 3).  
2. **Exponential Back‑off** – The delay before each retry follows `base_delay * 2^(attempt‑1)` (e.g., 100 ms, 200 ms, 400 ms) and is capped at 5 seconds.  
3. **Configurable Max Retries** – The environment variable `MAX_RETRIES` overrides the default; values < 0 are treated as 0 (no retries).  
4. **Logging** – Each retry logs: tool name, attempt number, error type, and error message in a structured JSON line.  
5. **Circuit Breaker** – After **5 consecutive failures** of the same tool, the circuit breaker opens, rejecting further calls for **60 seconds** and emitting a “circuit‑open” log entry.  
6. **Circuit Reset** – After the 60 s timeout the breaker automatically resets, allowing calls to proceed and resetting the failure counter.  
7. **Metrics** – Successful calls, retry attempts, and circuit‑breaker trips are exposed via Prometheus counters (`tool_calls_total`, `tool_calls_retried_total`, `circuit_breaker_trips_total`).

mod tests {
    use super::*;
    use std::sync::Arc;
    use tokio::time::{self, Duration};

    // Helper mock tool that can be programmed to fail a set number of times
    async fn mock_tool(fail_times: usize) -> Result<String, ToolError> {
        static CALL_COUNT: std::sync::Mutex<usize> = std::sync::Mutex::new(0);
        let mut cnt = CALL_COUNT.lock().unwrap();
        *cnt += 1;
        if *cnt <= fail_times {
            Err(ToolError::Transient("500 Internal".into()))
        } else {
            Ok("ok".into())
        }
    }

    #[tokio::test]
    async fn retries_up_to_max() {
        std::env::set_var("MAX_RETRIES", "2");
        let result = retry_tool_call(|| mock_tool(3)).await;
        assert!(result.is_err()); // exhausted retries
    }

    #[tokio::test]
    async fn succeeds_before_max_retries() {
        std::env::set_var("MAX_RETRIES", "5");
        let result = retry_tool_call(|| mock_tool(2)).await;
        assert_eq!(result.unwrap(), "ok");
        // verify that exactly 3 attempts were made (2 failures + 1 success)
    }

    #[tokio::test]
    async fn exponential_backoff_delays() {
        time::pause(); // freeze time
        let start = time::Instant::now();
        let _ = retry_tool_call(|| mock_tool(2)).await;
        // Expected delays: 100ms, 200ms (base 100ms)
        time::advance(Duration::from_millis(300)).await;
        assert_eq!(time::Instant::now() - start, Duration::from_millis(300));
    }

    #[tokio::test]
    async fn circuit_breaker_opens_after_five_failures() {
        let breaker = CircuitBreaker::new(5, Duration::from_secs(60));
        for _ in 0..5 {

        }
        assert!(breaker.is_open("tool_x"));
    }

        assert!(!breaker.is_open("tool_x"));
    }
