### 5. Request Flow

1. **Client** sends `POST /shield` with JSON body:
   ```json
   {
     "agent_id": "text_summarizer_v1",
     "user_id": "user-123",
     "payload": {"text": "..."},
     "config": {"max_length": 200}
   }
   ```
2. **Flask** parses body into `AgentRequest` (pydantic model).  
3. **Policy Engine** runs:
   - Schema validation (`jsonschema`).
   - RateŌĆælimit check (`redis` token bucket).
   - Quota check (`user_quota_db`).
   - Custom rules (e.g., no URLs in payload for `text_summarizer`).
4. If **approved**:
   - **Resource Guard** reserves CPU/memory (e.g., `docker run --cpus=0.5 --memory=256m`).
   - Request forwarded to **Agent Executor**.
   - Response returned to client.
5. If **denied**:
   - Return `403 Forbidden` with reason.
6. **Audit Logger** records decision, timestamps, and metrics.  
7. **Metrics Exporter** updates counters (`requests_total`, `denied_total`, `latency_seconds`).
